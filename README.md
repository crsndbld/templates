# templates

# Requirement
* git
* node.js
* next.js

# Installation
```bash
yarn create next-app --typescript
yarn add --dev --exact prettier
yarn add --dev eslint-config-prettier
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

# Usage
```bash
cd next-app
yarn dev

```
